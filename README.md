# addon-llama

Home Assistant addons for `ik_llama.cpp`, a performance-optimized fork of `llama.cpp` specifically tailored for ARM devices like the Raspberry Pi 5.

## Features

- **High Performance**: Native optimizations for ARMv8.2 (Pi 5).
- **Dynamic Configuration**: Configure model path, context size, and threads via the Home Assistant UI.
- **OpenAI Compatible**: Provides an OpenAI-compatible API endpoint on port 8080.

## Getting Started

See the [Documentation](llama/DOCS.md) for installation and configuration instructions.
